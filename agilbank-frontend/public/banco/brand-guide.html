Guia de Marca AgilBank

Identidade visual AgilBank. Arquivos oficiais PNG em agilbank-frontend/imagens/agilbank/; cópias de deploy em /brand/ (raiz estática).

Logo para fundo azul / escuro

Arquivo oficial: imagens/agilbank/logo agil bank clara.png (deploy: /brand/logo-agil-bank-clara.png).

<link rel="stylesheet" href="css/agilbank-brand.css" />
<!-- Header azul -->
<img src="/brand/logo-agil-bank-clara.png" alt="AgilBank"
     class="agilbank-brand-logo agilbank-brand-logo--header agilbank-brand-logo--dark"
     width="760" height="180" decoding="async" />

Logo para fundo branco / claro

Arquivo oficial: imagens/agilbank/logo agil bank escura.png (deploy: /brand/logo-agil-bank-escura.png).

<img src="/brand/logo-agil-bank-escura.png" alt="AgilBank"
     class="agilbank-brand-logo agilbank-brand-logo--light"
     width="760" height="180" decoding="async" />

Wordmark para cabeçalho mobile

Cabeçalhos estreitos (barra superior em telas mobile) ficam mais legíveis com apenas o texto “agilBank”, sem o ícone “A” à frente.

  • Header azul / escuro: /brand/logo-agil-bank-clara.png — texto branco / gradiente quase branco sobre fundo institucional.
  • Header ou topo claro: /brand/logo-agil-bank-escura.png — texto escuro para contraste sobre branco/off-white.

<!-- Barra azul -->
<img src="/brand/logo-agil-bank-clara.png" alt="AgilBank"
     class="agilbank-brand-wordmark-header"
     width="300" height="52" decoding="async" />

<!-- Fundo claro (ex.: modulos) — acrescentar --on-light -->
<img src="/brand/logo-agil-bank-escura.png" alt="AgilBank"
     class="agilbank-brand-wordmark-header agilbank-brand-wordmark-header--on-light"
     width="300" height="52" decoding="async" />
  • Usar wordmark sem ícone em headers pequenos.
  • Tamanhos: .agilbank-brand-wordmark-header limita altura/max-width quando a marca é tratada como faixa compacta.
  • Usar logo completa (PNG clara ou escura) em áreas com largura suficiente (hero, modais grandes, onboarding).
  • Usar ícone isolado oficial (icone 192m x 192m.png) só quando não houver espaço para texto — favicon, chips, splash.

Ícone isolado

imagens/agilbank/icone 192m x 192m.png (deploy: /brand/icone-192-oficial.png) — também base do PWA 192 px.

<img src="/brand/icone-192-oficial.png" alt=""
     class="agilbank-brand-icon" aria-hidden="true"
     width="145" height="140" decoding="async" />

Quando o ícone for só decorativo, use alt="" e aria-hidden="true". Se for o único elemento de marca no bloco, prefira alt="AgilBank" e não use aria-hidden.

Como importar

  1. Inclua a folha de estilo em cada HTML: <link rel="stylesheet" href="css/agilbank-brand.css" /> (caminho relativo a partir de public/banco/).
  2. Referencie o PNG certo para o contraste do fundo (regras abaixo).
  3. Mantenha proporção: use classes .agilbank-brand-logo / --header e não force largura com distorção.

Páginas que ainda apontam para assets/agilbank-brand.css continuam válidas: esse arquivo apenas reexporta css/agilbank-brand.css.

Regras de uso

  • Em header azul ou fundo escuro, usar /brand/logo-agil-bank-clara.png (origem logo agil bank clara.png).
  • Em fundo branco ou claro, usar /brand/logo-agil-bank-escura.png (origem logo agil bank escura.png).
  • Em espaços pequenos / PWA ícone / favicon, usar /brand/icone-192-oficial.png (origem icone 192m x 192m.png).
  • Não usar gov.br como marca do app.
  • Não distorcer a logo nem alterar proporção (CSS: height + width: auto ou classes fornecidas).
  • Evitar sombra pesada sobre a marca.
  • Não usar amarelo ou paleta de terceiros para substituir o gradiente institucional.
  • Não embutir base64 grande de marca no HTML; servir arquivo em /brand/.
  • Não usar wordmark escuro sobre header azul; não usar wordmark claro sobre fundo claro. Usar sempre o arquivo e a classe combinados conforme esta seção e .agilbank-brand-wordmark-header--on-light onde aplicável.
  • Manter alt="AgilBank" quando a imagem representar a marca textual completa.

Onde aplicar no app

  • index.html — barra superior azul: PNG claro /brand/logo-agil-bank-clara.png; login/splash/card claro: /brand/logo-agil-bank-escura.png.
  • deposito.html, transferir.html, movimentacao.html (faixa azul): /brand/logo-agil-bank-clara.png; card/branco onde aplicável: /brand/logo-agil-bank-escura.png.
  • modulos.html — topo claro + avatar: marca escura + ícone oficial 192 px conforme esta página.
  • manifest / ícones PWA — binários espelhos de icone 512m x 512m.png / icone 192m x 192m.png em public/banco/icons/.
  • emprestimos, notificações, reset-password, formulário… — alinhados ao mapa PNG acima.
