Guia de Marca AgilBank

Identidade visual para o frontend legado em public/banco/. Use sempre arquivos SVG externos — não embutir SVG grande no index.html.

Logo para fundo azul / escuro

Arquivo: assets/brand/agilbank-logo-dark-bg.svg — texto claro e gradiente no ícone “A”.

<link rel="stylesheet" href="css/agilbank-brand.css" />
<!-- Header azul -->
<img src="assets/brand/agilbank-logo-dark-bg.svg" alt="AgilBank"
     class="agilbank-brand-logo agilbank-brand-logo--header agilbank-brand-logo--dark"
     width="760" height="180" decoding="async" />

Logo para fundo branco / claro

Arquivo: assets/brand/agilbank-logo-light-bg.svg — tipografia escura e recorte interno claro.

<img src="assets/brand/agilbank-logo-light-bg.svg" alt="AgilBank"
     class="agilbank-brand-logo agilbank-brand-logo--light"
     width="760" height="180" decoding="async" />

Wordmark para cabeçalho mobile

Cabeçalhos estreitos (barra superior em telas mobile) ficam mais legíveis com apenas o texto “agilBank”, sem o ícone “A” à frente.

  • Header azul / escuro: assets/brand/agilbank-wordmark-header.svg — texto branco / gradiente quase branco sobre fundo institucional.
  • Header ou topo claro: assets/brand/agilbank-wordmark-header-on-light.svg — texto escuro para contraste sobre branco/off-white.

<!-- Barra azul -->
<img src="assets/brand/agilbank-wordmark-header.svg" alt="AgilBank"
     class="agilbank-brand-wordmark-header"
     width="300" height="52" decoding="async" />

<!-- Fundo claro (ex.: modulos) — acrescentar --on-light -->
<img src="assets/brand/agilbank-wordmark-header-on-light.svg" alt="AgilBank"
     class="agilbank-brand-wordmark-header agilbank-brand-wordmark-header--on-light"
     width="300" height="52" decoding="async" />
  • Usar wordmark sem ícone em headers pequenos.
  • Tamanhos (Fase Z.2): .agilbank-brand-wordmark-header — altura ~28px, largura até ~170px (com max-width relativo à viewport). Na barra azul do index.html, a mesma classe sobe para ~30px via .top-bar .agilbank-brand-wordmark-header.
  • Usar logo completa (dark-bg ou light-bg) em áreas com largura suficiente (hero, modais grandes, onboarding).
  • Usar ícone isolado só quando não houver espaço para texto nem para a composição inteira — favicon, avatar compacto decorativo em Serviços, splash, bolha compacta.

Ícone isolado

assets/brand/agilbank-icon.svg — favicon, avatar, splash ou espaços reduzidos.

<img src="assets/brand/agilbank-icon.svg" alt=""
     class="agilbank-brand-icon" aria-hidden="true"
     width="145" height="140" decoding="async" />

Quando o ícone for só decorativo, use alt="" e aria-hidden="true". Se for o único elemento de marca no bloco, prefira alt="AgilBank" e não use aria-hidden.

Como importar

  1. Inclua a folha de estilo em cada HTML: <link rel="stylesheet" href="css/agilbank-brand.css" /> (caminho relativo a partir de public/banco/).
  2. Referencie o SVG certo para o contraste do fundo (ver regras abaixo).
  3. Mantenha proporção: use classes .agilbank-brand-logo / --header e não force largura com distorção.

Páginas que ainda apontam para assets/agilbank-brand.css continuam válidas: esse arquivo apenas reexporta css/agilbank-brand.css.

Regras de uso

  • Em header azul ou fundo escuro, usar agilbank-logo-dark-bg.svg.
  • Em fundo branco ou claro, usar agilbank-logo-light-bg.svg.
  • Em espaços pequenos, usar agilbank-icon.svg.
  • Não usar gov.br como marca do app.
  • Não distorcer a logo nem alterar proporção (CSS: height + width: auto ou classes fornecidas).
  • Evitar sombra pesada sobre a marca.
  • Não usar amarelo ou paleta de terceiros para substituir o gradiente institucional.
  • Não colocar SVG grande inline no index.html.
  • Não usar wordmark escuro sobre header azul; não usar wordmark claro sobre fundo claro. Usar sempre o arquivo e a classe combinados conforme esta seção e .agilbank-brand-wordmark-header--on-light onde aplicável.
  • Manter alt="AgilBank" quando a imagem representar a marca textual completa.

Onde aplicar no app

  • index.html — barra superior azul: agilbank-wordmark-header.svg + .agilbank-brand-wordmark-header; login e demais áreas podem usar SVG em assets/ até migração completa.
  • deposito.html — header azul: agilbank-wordmark-header.svg + .agilbank-brand-wordmark-header.
  • transferir.html — headers de fluxo: mesmo wordmark sobre azul.
  • movimentacao.html — barra superior azul: wordmark; bloco/comprovante em card branco: agilbank-logo-light-bg.svg quando for logo completa no contexto claro.
  • modulos.html — topo claro: agilbank-wordmark-header-on-light.svg + .agilbank-brand-wordmark-header.agilbank-brand-wordmark-header--on-light.
  • emprestimos.html, notificacoes.html, reset-password, formulario… — alinhar na próxima iteração ao guia.
